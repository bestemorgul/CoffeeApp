# Coffee App

This app shows lists of the most preferred hot coffee types and iced coffe types including:

 MVVM,
 RecyclerView,
 Navigation,
 ViewModel,
 LiveData,
 DataBinding,
 Retrofit2,
 Coroutines,
 Moshi,
 Coil
 
 
  ## Features

- Images of hot coffe types & iced coffe types
- Short description for each coffee
- Ingredients of each coffe
